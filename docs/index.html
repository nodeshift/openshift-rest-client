openshiftClient

Builds the rest client based on provided or default kubernetes configuration.

openshiftClient(settings: object?): Promise
Parameters
settings (object? = {}) settings object for the openshiftClient function
Name Description
settings.url string? Openshift cluster url
settings.authUrl string? Openshift Basic auth url
settings.auth object? -
settings.auth.username string? username to authenticate to Openshift
settings.auth.password string? password to authenticate to Openshift
settings.insecureSkipTlsVerify boolean? flag to ignore TLS verification
Returns
Promise: Returns the Rest Client
